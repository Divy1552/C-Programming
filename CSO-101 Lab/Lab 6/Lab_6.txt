1. Count Number of Digits in a Number
Write a C program to count the number of digits in a given integer using a while loop.
Sample Input: Enter a number: 12345
Sample Output: Number of digits: 5

2. Check if a Number is Prime
Write a C program to check whether a given number is prime or not using a for loop.
Sample Input: Enter a number: 17
Sample Output: 17 is a prime number.

3. Print Fibonacci Series till nth term
Write a C program to print the Fibonacci series up to the nth term using a while loop.
Sample Input: Enter the number of terms: 6
Sample Output: Fibonacci Series: 0 1 1 2 3 5

4. Find Greatest Common Divisor (GCD)
Write a C program to find the GCD of two numbers using a while loop.
Sample Input: Enter two numbers: 36 60
Sample Output: GCD of 36 and 60 is 12
